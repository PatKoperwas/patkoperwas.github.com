Service for Scanning in Construction Drawings

Clients upload their construction set pdf (usually a few hundred pages and a few gigs in size), then this service splits it apart into individual sheets. To make data entry way easier we used OCR to pull out the text from each sheet and then do some fancy filtering to pull out useful metadata like the drawing name and number. You can see the auto-filled inputs in the screenshot.

JavaScript Based Markup Editor

A markup editor that allows construction professionals the ability to add markups to their construction drawings. Users can draw clouds, arrows, add text and create links to other sheets. Everything is synced to both web and mobile apps.

Documentation Site

I got tired of an out of date and not very pretty wiki. So I created a middleman application to replace it. Easy to update - each page has a quick link to the Github editor. On commit, the application rebuilds and updates itself automatically so edits can be viewed almost as soon as they are published.
